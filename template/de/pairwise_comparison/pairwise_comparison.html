Two-group differential gene expression

Jim Zhang

2016-06-16

Introduction This procedure performs a two-group comparison of all genes in a transcriptome data set. It accepts both signal intensities from microarray and read counts from RNA-seq. All genes must be annotated with NCBI Entrez unique gene IDs. The main elements of this procedure include:

  • Sample-level analysis
    • Overall distribution of gene expression level and between-sample variance
    • Sample similarity evaluated via sample-sample correlation, hierarchical clustering and principal components analysis
  • Gene-level analysis
    • Statistical test of group means on each genes. Call DeWrapper() to find available statistical tests, such as Significance Analysis of Microarrays for microarray data and edgeR for RNA-seq read count data
    • Visualization of differential gene expression through M-A plot, Volcano plot, heatmap, etc.
    • Selection of top-ranked differentially expressed genes (DEGs)
  • Gene set-level analysis
    • Over-representation analysis (ORA)
    • Gene set enrichment analysis (GSEA)
    • Visualization of KEGG pathway maps
    • Gene-gene set biclustering
